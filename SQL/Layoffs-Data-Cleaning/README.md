# Layoffs Data Cleaning Project

## Overview

This project focuses on cleaning a layoffs dataset using MySQL.

## Tasks Performed

- Removed duplicates
- Standardized country names
- Converted text dates to SQL DATE format
- Handled NULL values
- Removed unnecessary rows
- Removed helper columns

## SQL Concepts Used

- ROW_NUMBER()
- PARTITION BY
- CTEs
- DELETE
- UPDATE
- ALTER TABLE
- STR_TO_DATE()

## Tools

- MySQL
- MySQL Workbench
